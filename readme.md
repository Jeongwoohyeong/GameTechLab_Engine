# Week-02: Direct3D 11 기반 3D 씬 뷰어/편집기 샘플

Windows 및 Direct3D 11 기반으로 구현된 간단한 3D 씬(Scene) 뷰어 및 편집기 샘플 프로젝트입니다.

이 프로젝트는 ImGui를 사용하여 UI를 구성했으며, 사용자는 기본 프리미티브(큐브, 스피어, 트라이앵글, 플레인)를 생성하고 월드 내에서 선택, 이동, 스케일 조절을 할 수 있습니다. 렌더링된 객체는 마우스 피킹(Mouse Picking)을 통해 정밀하게 선택할 수 있으며, 기즈모(Gizmo)를 통해 직관적인 조작이 가능합니다.



---

## ✨ 주요 기능 (Features)

### 렌더링 (Rendering)
- **Direct3D 11**: D3D11 API를 기반으로 렌더링 파이프라인을 구축했습니다.
- **기본 셰이더**: 간단한 조명 및 색상 출력을 위한 버텍스/픽셀 셰이더를 포함합니다.
- **외곽선 렌더링**: 선택된 오브젝트를 시각적으로 강조하기 위해 외곽선(Outline)을 렌더링합니다.
- **월드 그리드 및 기즈모**: 씬의 공간감과 조작 편의성을 위해 월드 그리드와 로컬/월드 기즈모를 렌더링합니다.

### 카메라 (Camera)
- **모드 전환**: 원근(Perspective) 카메라와 직교(Orthographic) 카메라 모드를 전환할 수 있습니다.
- **자유로운 조작**: 마우스 우클릭 드래그로 카메라를 회전(Pitch/Yaw)하고, `W/A/S/D` 키로 월드 공간을 자유롭게 이동할 수 있습니다.

### 프리미티브 (Primitives)
- **4종 프리미티브**: **큐브(Cube), 스피어(Sphere), 트라이앵글(Triangle), 플레인(Plane)** 생성을 지원합니다.
- **객체 관리**: 생성된 모든 프리미티브는 고유한 UUID(Universally Unique Identifier)를 통해 관리됩니다.

### 기즈모 (Gizmo)
- **3종 조작**: 축 기반 **이동(Translate), 스케일(Scale), 회전(Rotate)** 기즈모를 지원합니다. (단, 회전은 일부만 구현)
- **좌표계 지원**: 기즈모는 **로컬(Local) 모드**와 **월드(World) 모드**를 모두 지원하여 정밀한 변형이 가능합니다.

### 상호작용 (Interaction)
- **마우스 피킹**: 마우스 좌클릭으로 3D 월드 내의 오브젝트 또는 기즈모의 특정 축을 정확하게 선택할 수 있습니다.

### 씬 관리 (Scene Management)
- **파일 I/O**: 씬을 **JSON 형식(.Scene)** 파일로 저장하고 불러올 수 있습니다. (새 씬, 저장, 불러오기)

### UI (User Interface)
- **ImGui 기반 패널**: 모든 기능은 ImGui를 통해 구현된 패널에서 제어할 수 있습니다.
  - **Control Panel**: FPS 정보, 프리미티브 생성, 씬 관리, 카메라 파라미터 제어.
  - **Property Window**: 선택된 객체의 Transform(위치, 회전, 크기) 정보 표시 및 수정.
  - **Console**: 예제 로그 및 디버깅 정보를 출력하는 콘솔 창.

## 🎮 조작법 (Controls)

- **카메라 회전**: `마우스 우클릭 + 드래그`
- **카메라 이동**: `W`, `A`, `S`, `D` 키
- **오브젝트/기즈모 선택**: `마우스 좌클릭`
  - 기즈모 축 위에서 클릭 시 해당 축이 선택됩니다(노란색으로 강조).
  - 빈 공간이나 오브젝트를 클릭하면 해당 오브젝트가 선택됩니다.
- **기즈모 조작**: `선택된 축 위에서 마우스 좌클릭 + 드래그`
  - **이동 (Translate)**: 선택된 축 방향으로 오브젝트를 이동합니다.
  - **스케일 (Scale)**: 선택된 축 방향으로 오브젝트의 크기를 조절합니다.
  - **회전 (Rotate)**: 현재 구현 중입니다.

## 🔧 빌드 및 실행 방법 (Getting Started)

### 사전 준비 (Prerequisites)
- **OS**: Windows 10 / 11 (x64)
- **IDE**: Visual Studio 2022 (`C++ 데스크톱 개발` 워크로드 설치 필요)
- **SDK**: Windows SDK (최신 버전, Direct3D 11 포함)

### 빌드 순서 (Build & Run)
1.  `Week-02.sln` 파일을 Visual Studio 2022에서 엽니다.
2.  솔루션 구성을 `Debug` 또는 `Release`, 플랫폼을 `x64`로 설정합니다.
3.  `빌드` 메뉴에서 `솔루션 빌드`를 선택하여 컴파일을 진행합니다.
4.  빌드가 완료되면 `F5` (디버깅 시작) 또는 `Ctrl+F5` (디버깅 없이 시작) 키를 눌러 프로그램을 실행합니다.
    -   실행 파일은 `x64/Debug/` 또는 `x64/Release/` 폴더에 생성됩니다.

## 📄 씬 저장 형식 (.Scene)

씬 파일은 프로젝트 루트의 `Scenes/` 폴더에 **JSON** 형식으로 저장됩니다. (폴더가 없으면 자동 생성)

-   **파일명**: `<입력한_씬_이름>.Scene` (예: `Scenes/MyScene.Scene`)
-   **JSON 구조**:
    ```json
    {
        "Version": 1,
        "NextUUID": 5,
        "Primitives": {
            "1": {
                "Location": [1.5, 0.0, -2.0],
                "Rotation": [0.0, 0.0, 0.0, 1.0],
                "Scale": [1.0, 1.0, 1.0],
                "Type": "Cube"
            },
            "2": {
                "Location": [-1.0, 1.0, 0.0],
                "Rotation": [0.0, 0.0, 0.0, 1.0],
                "Scale": [0.5, 0.5, 0.5],
                "Type": "Sphere"
            }
        }
    }
    ```

## 🏗️ 코드 구조 (Code Structure)

-   `Week-02/main.cpp`: 프로그램의 시작점. 윈도우 생성, 메시지 루프, 렌더러/씬 초기화 및 피킹 로직 처리.
-   `URenderer.(h/cpp)`: 렌더링 총괄. D3D 리소스 생성, 씬/기즈모/UI 렌더링, 외곽선 처리.
-   `UD3dDevice.(h/cpp)`: D3D11 디바이스, 스왑체인 등 핵심 인터페이스 관리.
-   `UCamera.(h/cpp)`: View/Projection 행렬 계산, 카메라 이동/회전, Ray 생성 유틸리티.
-   `CInputManager.(h/cpp)`: 키보드 및 마우스 입력 상태 관리.
-   `CScene.(h/cpp)`: 프리미티브 객체들의 생성, 관리, 소멸을 담당.
-   `UPrimitiveComponent.*`: 프리미티브의 기본 클래스 및 파생 클래스(Cube, Sphere 등).
-   `LocalGizmo.(h/cpp)`: 기즈모의 렌더링, 피킹, 드래그 로직 처리.
-   `Math/`: FTransform, FRay 등 수학 관련 유틸리티 클래스 및 함수.
-   `External/`: 외부 라이브러리 (ImGui, SimpleJSON).

## 📝 알려진 제한 및 메모 (Known Limitations)

-   **기즈모 회전**: 회전(Rotate) 기능은 UI 및 기본 로직 골격만 포함되어 있으며, 실제 회전 변환 로직은 추가 구현이 필요합니다.
-   **기즈모 모드 전환**: 이동/스케일/회전 모드(`GizmoSwitch` 변수)는 현재 UI로 제어되지 않으며, 필요 시 UI 버튼 등을 추가하여 확장할 수 있습니다.
